# CipherSQLStudio

CipherSQLStudio is a web-based SQL learning platform. Users can view assignments, write SQL queries, and run them against a PostgreSQL database. The app provides AI-generated hints to guide learning without giving full solutions.  

**Features:**  
- Assignment list with title, description, difficulty  
- SQL editor with real-time results  
- AI hints for guidance  
- Mobile-friendly responsive design  

**Tech Stack:** React.js, SCSS, Node.js, Express.js, PostgreSQL, OpenAI GPT API  

**Setup:**  
Clone the repo, install dependencies, configure `.env`, run backend (`node server.js`) and frontend (`npm start`).
